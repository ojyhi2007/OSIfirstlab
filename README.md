# Первая лабораторная по предмету "Операционные системы". Выполнил: Мельников Илья Никитович

Репозиторий содержит две программы для вывода системной информации:

sys-info-win.py — версия для Windows

sys-info-linux.sh — версия для Linux

Каждая программа использует нативные механизмы своей операционной системы для получения сведений о системе.

Windows: sys-info-win.py

Назначение: вывод сведений о системе Windows.

Используемые технологии:

WinAPI через модуль ctypes

Запрос системной информации из DLL:

kernel32.dll

advapi32.dll

psapi.dll

ntdll.dll

Получаемые данные:

версия ОС

имя компьютера

имя пользователя

архитектура процессора

количество логических процессоров

физическая и виртуальная память

файл подкачки

список логических дисков

Linux: sys-info-linux.sh

Назначение: вывод сведений о системе Linux.

Используемые источники:

/proc

/etc/os-release

uname

nproc

statvfs

Получаемые данные:

дистрибутив и версия ядра

имя пользователя и hostname

архитектура системы

количество логических процессоров

оперативная память и swap

виртуальная память

средняя загрузка системы

список файловых систем и объём дисков
